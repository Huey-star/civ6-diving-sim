## 📖 简介 (Introduction)

你是否曾经打开《文明6》，兴冲冲地准备大干一场，结果发现出生在没有淡水的雪地？
你是否有一个晚上玩了4个小时，其中3小时50分钟都在点击“重新开始”？
你是否对肖恩·宾（Sean Bean）的那句开场白产生了严重的 PTSD？

**如果是，那么这款模拟器就是为你量身定做的。**

这是一个基于 Web 的极简模拟器，完美还原了《文明6》玩家最核心的玩法——**刷图（跳水）**。它能够模拟各种离谱的出生地、不可预测的结局，以及那些只有老玩家才懂的“血压时刻”。

## ✨ 版本特性 (Features V5.0)

本次更新为 **V5.0 巴比伦飞天版**，包含以下核心功能：

* **沉浸式体验**：完美复刻游戏加载时的深沉旁白与等待焦虑感。
* **真实随机性**：从 F 级雪地烂图到 S 级帕依提提神图，一切全看脸。
* **多结局系统**：
    * ✅ **科技/文化/征服胜利**：如果你能苟住的话。
    * ❌ **花式暴毙**：被蛮族骑脸、被甘地核平、被维苏威火山活埋。
    * 🖥️ **打破第四面墙**：因显卡爆炸或天亮上班而被迫退游。
* **特殊剧本 (Scenario System)**：
    * 🛠️ **时空特异点**：巴比伦 + 沿河丘陵 = 公元前手搓双翼机。
    * 🧱 **大建奇观**：秦始皇 + 沙漠 = 奇观误国 vs 奇观兴邦。
    * 💰 **黄金国度**：马里 + 沙漠 = 有钱没命花。
    * ❄️ **核动力圣地**：俄罗斯 + 冻土 = 生产力爆炸。
